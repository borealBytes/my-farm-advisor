# Offline Appliance

Offline appliance is the future stronger delivery mode for artifacts that want local-system-like users, services, jobs, and permissions.

## 🖥️ Purpose

Use this mode when the desired experience is closer to a tiny local appliance than a plain HTML document.

Examples:

- desktop-like delivery shell
- background job and cron-style delivery
- service principals that generate user-visible outputs
- stronger permission semantics than normal browser delivery can provide

## 🧭 Status

This mode is spec-first. It defines the boundary between plain offline HTML and a future runtime-backed system.
