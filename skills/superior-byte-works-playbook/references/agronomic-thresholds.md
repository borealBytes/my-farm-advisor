# Reference: Agronomic Action Thresholds

For use in incident-response playbooks. Thresholds are defaults
for Upper Michigan / Great Lakes region. Override with local
extension recommendations where available.

## Pest Thresholds (Corn)

| Pest | Scouting Method | Action Threshold | Notes |
|------|----------------|-----------------|-------|
| Corn Rootworm (adult) | Yellow sticky traps | 5 beetles / trap / day | Pre-harvest, silk protection |
| European Corn Borer | Egg mass counts | 1 egg mass / 3 plants | At whorl stage |
| Aphids (corn leaf) | Colony counts | > 400 aphids / plant | Pre-tassel; predator presence matters |
| Wireworm | Bait station | > 1 larva / bait station | Pre-plant; replant threshold |
| Western Bean Cutworm | Pheromone trap | 8+ moths / night (2 cons. nights) | V14–R1 stage |

## Disease Thresholds (Corn)

| Disease | ID Criteria | Action Threshold | Notes |
|---------|------------|-----------------|-------|
| Gray Leaf Spot | Lesions on 3rd leaf below ear | Present at or before silking | Fungicide if susceptible hybrid |
| Northern Corn Leaf Blight | AUDPC trending up | Lesions on ear leaf at silking | Economic only with susceptible hybrid |
| Goss's Wilt | Freckles + watersoaking | Any confirmed presence | No treatment; report to extension |

## Weed Thresholds

| Weed Type | Threshold | Timing |
|-----------|-----------|--------|
| Broadleaf weeds | > 2 plants / sq ft | Pre-V4 corn |
| Grasses (general) | > 3 plants / sq ft | Pre-V4 corn |
| Volunteer corn | Any presence | Pre-plant or V1–V2 |

## Nutrient Deficiency Visual ID

| Deficiency | Visual Sign | Confirmation | Response |
|------------|-------------|--------------|----------|
| Nitrogen | Yellowing from leaf tip, lower leaves first | PSNT soil test | Side-dress N |
| Potassium | Marginal leaf scorch, lower leaves | Tissue + soil test | In-season K foliar (limited efficacy) |
| Sulfur | Interveinal yellowing, upper leaves | Tissue test | 10–15 lb/ac S side-dress |
| Zinc | White striping near midrib, young leaves | Tissue test | 1 lb/ac Zn foliar |
